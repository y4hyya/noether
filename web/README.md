# Noether Web Frontend

Next.js frontend for the Noether Perpetual DEX.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Wallet integration with Freighter
- Trading interface
- Position management
- Liquidity provider interface

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Freighter Wallet SDK
- Stellar SDK
